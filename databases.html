Database Fundamentals

SQL (Structured Query Language) is a standard database language that is used to create, maintain, and retrieve relational databases. Advanced programming and web development students should be familiar with SQL as an essential part of "back end" (server-side) programming skills for creating apps and websites. Database engineering is an entire job, a combination of more traditional systems administration and software development. Almost every programmer will need to use a form of SQL at some point in their careers.

These lessons can be done with SQLite on Repl.it. The goal is for students to be able to pass the LinkedIn T-SQL certification exam. If students have access to Visual Studio, it would be ideal for them to practice writing T-SQL code directly with it. Each week contains slides, short challenges, answers to the challenges, and a project with a rubric for students to complete creatively and independently.

This six-week course covers the vocational frameworks:

  1. Explain database concepts
  2. Explain the purpose of a relational database
  3. Demonstrate the planning and designing of a relational database
  4. Classify the relationship between tables such as one-to-one and one-to-many
  5. Give examples of database queries and data reports
