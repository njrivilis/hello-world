Fundamentals of Computer Science

This course is designed to follow either the Unity Introduction to C# or the AP Computer Science Principles courses. It covers professional/college level object oriented programming material, quizzes, and projects. Students will learn data types, logic and conditionals, methods, arrays, loops, classes, interfaces, inheritance, and references. The course begins with C#, with the hope that students have some familiarity with it from Unity, and then reviews the same concepts with Python, so that students can create their final projects using Python and JavaScript libraries.
