This flowchart illustrates paths between various options for computer science, programming, and web development classes.

Any of these courses can be done in isolation, but to give students appropriate context for each aspect of computer science and web development, it is best for them to take an introductory level course first, such as web design or computer science fundamentals, before proceeding to more "special topics" classes on game design, object oriended programming, or databases. The advanced web design and college and career readiness material is designed for a junior or senior level class, with information about professional tools and skills needed to succeed in the field.

I do not currently teach AP Computer Science A, because I do not believe that learning Java is in the best interest of most students. C# is sometimes called "Java done right" and I have had great success with teaching it, especially with the Unity environment for game development. If desired, I would be open to writing Java-based AP Computer Science A curriculum to be taught on Mac or Linux, but if my students have access to Windows PCs, I think that C# is the better choice to cover object oriented fundamentals. My hope is that in the near future, the AP exam will switch to Python, which many schools and colleges are starting to favor. I've found that Python is easier to pick up after learning C# than vice versa, but would consider creating and teaching a Python-only course if there was high interest from students and staff.
