Advanced Web Design: Fullstack Skills

This course is for advanced students who will learn everything they need to know to create beautiful responsive websites from scratch using JavaScript and NodeJs. It reviews fundamentals of HTML, CSS, and JavaScript in greater depth, with more independent projects and exercises. Along the way students will learn Git and deployment so they can build their own portfolio, collaborate with others, and see their work on the web.

Students should be comfortable creating static web pages, using HTML tags, and designing CSS stylesheets before taking this course. Right now, the course is focused on leading into the Node.js open source server environment. The object oriented programming course covers C# and Python fundamentals, but (as of right now) this module is entirely focused on JavaScript. Similar courses might cover jQuery, but it is fast becoming obsolete and I do not currently teach it.
